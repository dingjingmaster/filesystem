## 1. 修改内容

- 打开方式弹窗“设为默认打开方式”改为固定尺寸的自绘勾选按钮，选中和未选中状态都绘制实色背景，避免取消选中时依赖空文本清理旧对钩。
- 打开方式应用列表右侧选中标记改为同一固定尺寸勾选指示器，选中时绘制 SVG 对钩，未选中时绘制同尺寸空白实色区域。
- 修正通用 checkbox 样式的状态判断，`Hovered` 和 `Disabled` checked 状态也按 checked 绘制，提高对钩可见性。
- 补充 `OpenWithSetDefault(false)` 状态切换测试。
